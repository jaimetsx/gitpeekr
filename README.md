# GitPeekr

GitPeekr is a web page for exploring public GitHub profiles. Type a username and it shows a quick summary of that account, their public contribution activity, and their most popular repositories.

## What it shows

- Profile summary with avatar, bio, location, company, website, and account stats.
- A contribution heatmap for public activity across available years.
- Top repositories with stars, forks, language, topics, and last update time.
- Direct links back to the GitHub profile and each repository.

## How To Use

1. Type any public GitHub username into the search box on the home page.
2. Press Enter or click Search.
3. Use the example usernames if you want to try it fast.
4. On the profile page, switch the year on the contribution chart to see activity for different years.
5. Click a repository card to open that project on GitHub.
